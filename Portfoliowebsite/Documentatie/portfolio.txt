voorbeelden voor analyses:

ML:
- titanic op kaggle.com
- human activity recognition using smartphone dataset: 
	http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones
- Hubway visualization challenge: 
	http://hubwaydatachallenge.org/trip-history-data/
- Movielens data: I couldn’t have left this data set out. Bigger than some of the other data sets mentioned in the article, but provides a lot of fun. The dataset is sufficient to build a recommender system and see which movies are liked by what kind of audience.
	Readme file: http://files.grouplens.org/papers/ml-1m-README.txt
	Dataset: http://www.grouplens.org/system/files/ml-1m.zip
- dataset met alle automodellen: A lot of correlation, exploration, visualizations are possible, linking fuel type, with mpg, performance improvement over various car models, emission ratings, etc.
	http://fueleconomy.gov/feg/download.shtml
